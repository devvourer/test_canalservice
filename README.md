## ⚙ Запуск:

Создать файл `.env` в директории проекта с содержимым

`- DB_NAME=postgres`

`- DB_PASSWORD=1`

`- DB_PORT=5432`

`- DB_USER=postgres`

`- DB_HOST=db`

Запустить команду: 

`docker-compose up -d --build`
